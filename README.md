# hackathon
The ReactEurope hackathon.
News Reader using touchstonejs, flask, redis, reddit and twitter apis 

## Installation

Go to public/ and 
```
npm install
```
```
npm start
```

in the root directory start the Flask server with 

```
python main.py
```

NOTE : Requires Python 2.7+ and a running redis-server
